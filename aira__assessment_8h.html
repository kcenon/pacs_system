Detailed Description

IHE AIRA (AI Result Assessment) - Assessment Creator Actor.

Implements the IHE AIRA profile for creating structured assessments of AI-generated results. Clinicians can Accept, Modify, or Reject AI results, and this assessment is encoded as a DICOM Structured Report linked to the original AI result.

See also
IHE RAD AIRA Supplement (2025-06-12)
DICOM PS3.3 - Enhanced SR IOD
