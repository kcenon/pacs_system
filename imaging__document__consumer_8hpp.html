Detailed Description

IHE XDS-I.b Imaging Document Consumer Actor.

Provides the Imaging Document Consumer actor for the IHE XDS-I.b integration profile. This actor queries the XDS registry for imaging documents and retrieves referenced images via WADO-RS or C-MOVE.
