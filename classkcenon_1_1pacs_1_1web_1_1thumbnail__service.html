Detailed Description

Thumbnail generation and caching service.

Generates thumbnails from DICOM images with server-side caching. Supports multiple output formats and sizes.

Example
auto service = std::make_shared<thumbnail_service>(database);
thumbnail_params params;
params.size = 256;
params.format = "jpeg";
params.quality = 75;
auto result = service->get_thumbnail("1.2.3.4.5", params);
if (result.success) {
// Use result.entry.data and result.entry.content_type
}
