Detailed Description

Validator for RT Dose IODs.

Validates DICOM datasets against the RT Dose IOD specification. Checks dose grid attributes, units, and summation type.

Validated Modules

Mandatory Modules

  • Patient Module (M)
  • General Study Module (M)
  • RT Series Module (M)
  • Frame of Reference Module (M)
  • General Equipment Module (M)
  • General Image Module (M)
  • Image Plane Module (C)
  • Image Pixel Module (C)
  • Multi-frame Module (C)
  • RT Dose Module (M)
  • RT DVH Module (U)
  • SOP Common Module (M)
