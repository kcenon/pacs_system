Detailed Description

Validator for RT Structure Set IODs.

Validates DICOM datasets against the RT Structure Set IOD specification. Checks ROI definitions, contours, and referenced frame of reference.

Validated Modules

Mandatory Modules

  • Patient Module (M)
  • General Study Module (M)
  • RT Series Module (M)
  • General Equipment Module (M)
  • Structure Set Module (M)
  • ROI Contour Module (M)
  • RT ROI Observations Module (M)
  • Approval Module (U)
  • SOP Common Module (M)
