Detailed Description

Nuclear Medicine (NM) Image IOD Validator.

Provides validation for Nuclear Medicine Image Information Object Definitions as specified in DICOM PS3.3 Section A.5 (NM Image IOD).

See also
DICOM PS3.3 Section A.5 - NM Image IOD
