Validator for MR Image IODs.

Validator for MR Image IODsValidates DICOM datasets against the MR Image IOD specification (PS3.3 Section A.4).

Validated Modules (PS3.3 Table A.4-1)

Mandatory Modules

  • Patient Module (M)
  • General Study Module (M)
  • General Series Module (M)
  • Frame of Reference Module (M)
  • General Equipment Module (M)
  • General Image Module (M)
  • Image Pixel Module (M)
  • MR Image Module (M)
  • SOP Common Module (M)
