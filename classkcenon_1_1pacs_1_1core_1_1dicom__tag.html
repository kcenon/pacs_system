Extract the Private Creator block number from this tag.

For a private data element (gggg,xxyy) where xx >= 0x10, returns xx. Per DICOM PS3.5 §7.8.1, the high byte of the element number identifies which Private Creator block owns this data element.

The block number (0x10-0xFF), or nullopt if not a private data element
