Detailed Description

C++20 module partition for IOD validators.

This module partition exports IOD (Information Object Definition) validators for various DICOM modalities:

  • DX: Digital X-Ray
  • MG: Mammography
  • NM: Nuclear Medicine
  • PET: Positron Emission Tomography
  • RT: Radiation Therapy
  • SEG: Segmentation
  • SR: Structured Reporting
  • US: Ultrasound
  • XA: X-Ray Angiography

Part of the kcenon.pacs module.
