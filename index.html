System Overview

PACS System is a modern C++20 PACS (Picture Archiving and Communication System) library for medical imaging. It implements the DICOM standard from scratch with zero external DICOM library dependency, providing complete SOP class support, transfer syntax handling, and DICOMweb services.

Built on the kcenon ecosystem infrastructure (networking, threading, logging, containers), PACS System delivers a layered architecture of 12 library targets covering the full DICOM pipeline from tag-level data handling through network services to cloud storage and AI integration.

Quick Start

A minimal example reading a DICOM file and extracting patient information:

#include "kcenon/pacs/core/dicom_file.h"
#include "kcenon/pacs/core/dicom_dataset.h"
#include "kcenon/pacs/core/dicom_tag_constants.h"
#include <iostream>
int main() {
// Read a DICOM Part 10 file
auto result = kcenon::pacs::core::dicom_file::read("image.dcm");
if (!result.is_ok()) {
std::cerr << "Failed to read: " << result.error().message << "\n";
return 1;
}
auto& file = result.value();
auto& dataset = file.dataset();
// Extract patient metadata
auto patient_name = dataset.get_string(kcenon::pacs::core::tags::patient_name);
auto study_date = dataset.get_string(kcenon::pacs::core::tags::study_date);
std::cout << "Patient: " << patient_name.value_or("(unknown)") << "\n";
std::cout << "Study: " << study_date.value_or("(unknown)") << "\n";
return 0;
}

Installation

CMake FetchContent (Recommended)

include(FetchContent)
FetchContent_Declare(
pacs_system
GIT_REPOSITORY https://github.com/kcenon/pacs_system.git
GIT_TAG v0.1.0
)
FetchContent_MakeAvailable(pacs_system)
target_link_libraries(your_target PRIVATE
kcenon::pacs_core
kcenon::pacs_encoding
kcenon::pacs_services
)
Note
PACS System requires DCMTK-compatible libraries and ICU for character set support. SQLite3 >= 3.45.1 is needed for storage features. Optional codec libraries (libjpeg-turbo, OpenJPEG, CharLS, OpenJPH) enable additional transfer syntax support.

vcpkg

vcpkg install kcenon-pacs-system
find_package(pacs_system CONFIG REQUIRED)
target_link_libraries(your_target PRIVATE kcenon::pacs_core)
