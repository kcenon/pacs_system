Validator for Ophthalmic Photography and Tomography Image IODs.

Validator for Ophthalmic Photography and Tomography Image IODsValidates DICOM datasets against the Ophthalmic Photography and Tomography Image IOD specifications (PS3.3 Sections A.39/A.40).

Validated Modules

Mandatory Modules

  • Patient Module (M)
  • General Study Module (M)
  • General Series Module (M) — Modality = "OP" or "OPT"
  • Ophthalmic Photography Image Module (M)
  • Image Pixel Module (M)
  • SOP Common Module (M)
  • Acquisition Context Module (M)

Conditional Modules

  • Multi-frame Module (C) — for OCT images
ophthalmic_iod_validator validator;
auto result = validator.validate(dataset);
if (!result.is_valid) {
for (const auto& finding : result.findings) {
std::cerr << finding.message << "\n";
}
